1. 项目简介
  - 主要用来对柠檬班-前程贷的注册，登录，充值，提现接口做自动化测试

2. 环境配置
  - 需安装python3.5以上版本
  - 需要安装openpyxl，PYYAML，requests等第三方库

3. 项目组成
  - common：存放一个封装的公共方法
  - config：里面是配置文件
  - data：测试用例，excel表
  - libs：重写的方法等
  - log：存放运行完后的日志
  - report：存放运行完后的测试报告
  - test_cases:存放测试的类
  - run.py： 运行文件
